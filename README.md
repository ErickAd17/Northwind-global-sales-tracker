# Northwind-global-sales-tracker
This project utilizes Power BI to transform Northwind Traders' sales data into actionable business insights. The primary goal is to provide an interactive tool to track Total Sales performance and visualize exactly where customers are located globally, enabling stakeholders to identify key market clusters and regional growth opportunities.

Key Features
Geospatial Tracking: An interactive map visual highlighting sales distribution across global regions.
Core Business KPIs: Real-time visibility of Total Revenue, Order Volume, and Average Order Value.
Time-Intelligence Analysis: Filters to compare sales performance across different years, quarters, and months.
Product Performance: Breakdown of sales by category to identify top-performing inventory.
